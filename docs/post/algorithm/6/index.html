【算法系列】生成窗口最大值数组

2022年10月1日
| algorithm

题目要求

有一个整形数组arr和一个大小为w的窗口从数组最左边滑到最右边，窗口每次向右边滑一个位置。例如，数组为[4,3,5,4,3,3,6,7], 窗口大小为3时:

如果数组长度为n，窗口大小为w，则一共产生 n-w+1 个窗口最大值。请实现一个函数：

  1. 输入：整形数组arr，窗口大小为w
  2. 输出：一个长度为 n-w+1 的数组res，res[i]表示每一种窗口状态下的最大值，要求时间复杂度O(N*w)

解题思路

整体逻辑：

  1. 创建一个双端数组，我们需要根据不同情况从两端分别push和pop
  2. 核心就是循环这个arr，把最大的值在arr中的下标放到双端数组的最左端(对应下文程序中"for !bq.Empty() && arr[i] > arr[bq.Back().(int)] {}“的处理)
  3. 当窗口走过对应数值区域的时候，把相应的数据进行过期(对应下文程序中"if bq.Front().(int) == i-w {}")
  4. 注意，因为要输出长度为 n-w+1, 所以循环当"i >= w -1"的时候，才开始输出结果到res中

golang实现

package main

func Getmaxwindow(arr []int, w int) (res []int) {
	res = []int{}
	// 1. create a bidirectional queue to store the result
	bq := newCustomQueue()
	// 2. for i < length(arr)
	for i := 0; i < len(arr); i++ {
		// 3. for !empty or value(in queue) > value(in bqueue), popback bqueue
		for !bq.Empty() && arr[i] > arr[bq.Back().(int)] {
			bq.PopBack()
		}
		// 4. pushback id(in queue)
		bq.Pushback(i)
		// 5. if bqueue's id >= i - w, expire the front value(in bqueue)
		if bq.Front().(int) == i-w {
			bq.PopFront()
		}
		// 6. record results to res slice
		if i >= w-1 {
			res = append(res, arr[bq.Front().(int)])
		}
	}
	return res
}

测试用例

package main

import (
	"github.com/stretchr/testify/assert"
	"testing"
)

func TestGetmaxwindow(t *testing.T) {
	tests := []struct {
		// 名字
		name string

		// 输入部分
		arr []int
		w   int

		// 输出部分
		wantRes []int
	}{
		{
			name:    "getmaxwindow",
			arr:     []int{4, 3, 5, 4, 3, 3, 6, 7},
			w:       3,
			wantRes: []int{5, 5, 5, 4, 6, 7},
		},
		{
			name:    "getmaxwindow1",
			arr:     []int{4, 3, 5, 4, 3, 3, 6, 7},
			w:       4,
			wantRes: []int{5, 5, 5, 6, 7},
		},
		{
			name:    "getmaxwindow2",
			arr:     []int{4, 3, 9, 8, 3, 1, 6, 7},
			w:       3,
			wantRes: []int{9, 9, 9, 8, 6, 7},
		},
	}
	for _, tt := range tests {
		t.Run(tt.name, func(t *testing.T) {
			// test case
			res := Getmaxwindow(tt.arr, tt.w)
			assert.Equal(t, tt.wantRes, res)
		})
	}
}
