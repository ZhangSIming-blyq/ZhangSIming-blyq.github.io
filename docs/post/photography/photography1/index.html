【摄影入门系列】人像摄影

摄影基础概念

对数码相机而言，影像的形成是由入射光投射在CMOS感光元件（全 称：互补式金属氧化物半导体）上，转变成强度不一的电流信号，再由相机处理器转换成可识别的图像文件，并加以存储的过程。

  1. 光圈：光圈是镜头上一个控制通光孔大小的装置，可以根据设定改变孔径大小。摄影中经常提到的光圈值(f值)即与此 装置有关。
    • f值越大，光圈越小；f值越小，光圈越大
    • 更大的光圈还能造成漂亮的虚化
  2. 快门: 我们把快门开启到关闭这段时间称为快门速度（有时候也直接简称为“快门”）。实际上也就是CMOS的感光时间
    • 快门时间过长，会导致进入的光太多，可能造成“曝光过 度”；而快门时间过短，则可能造成“曝光不足”
    • 较短的快门速度有助于凝固运动物体

这个规律就叫“倒易率”，在曝光中非常有用，为实现景深控制和运动物体清晰度控制提供了很大的便利。 譬如你想拍风景，在f/4.0下对应的快门速度是1/500s，但拍风光需要小景深，这时候缩小三挡光圈，变为f/11 （f/4.0→f/5.6→f/8→f/11），快门速度则相应延长三挡，变为1/60s（1/500s→1/250s→1/125s→1/60s），这两个 参数组合对曝光的影响是一致的，可以拍出相同亮度的照片。但是后一种组合方式可以得到更大景深，而同时曝光 也依然准确

  1. 感光度（ISO ）: 通常来讲，低感光度指ISO 200以下的设定，高感光度则指ISO 400以上的设定。你可以先尝试在光线充足的时 候，比如白天的室外，应该用尽量低的ISO。而在光线条件不足的情况下，应该设定更高的ISO

曝光组合: 光圈，快门，ISO

直方图: 直方图是一种图形，它可以显示照片中各个亮度级别的像素数量。直方图的横轴表示亮度级别，从左到右依次为暗到亮，纵轴表示该亮度级别的像素数量。直方图的形状可以帮助你判断照片的曝光是否正确。

  1. 防抖功能: 镜头防抖功能是在镜头中加入一片补偿镜片，该镜片可以在相机抖动时进行反向补偿
  2. 焦点: 焦点是指镜头所能够成像的距离范围，也就是镜头的成像距离。焦点越短，镜头所能够成像的距离范围就越小，焦点越长，镜头所能够成像的距离范围就越大
    • 半按快门按钮，即可对被摄物体实现对焦
  3. 景深控制:
    • 光圈 光圈越大，景深越小；光圈越小，景深越大。
    • 焦距 焦距越长，景深越小；焦距越短，景深越大。
    • 对焦距离 对焦距离越远，景深越大；对焦距离越近，景深越小。
    • 画幅 画幅越小，景深越大；画幅越大，景深越小。
  4. 白平衡：白平衡是指相机根据不同的光源，调整相机的色温，使得白色物体在不同光源下都能保持白色。白平衡的设定可以让照片的色彩更加准确，更加真实。
    • 使用白平衡去还原真实肉眼看到的场景

人像摄影

参数模式

在人像摄影中，正确地设置相机参数是非常关键的。这些参数包括模式、快门速度、光圈、ISO、景深、闪光灯和焦距。下面我将分别解释每个参数的设置以及如何根据不同情况进行调整：

  1. 模式 (Mode):

    • 设置: 对于初学者，可以选择Aperture Priority (光圈优先) 模式，这样你只需调整光圈大小，相机会自动设定其他参数。更有经验的摄影师可能会选择Manual (手动) 模式，以完全控制所有设置。

  2. 快门速度 (Shutter Speed):

    • 设置: 快门速度应根据拍摄环境和需要达到的效果来调整。一般来说，确保至少在1/60秒或更快，以避免因手抖而模糊。如果拍摄动态人像，可能需要更快的快门速度。

  3. 光圈 (Aperture):

    • 设置: 光圈控制景深和进入镜头的光量。在人像摄影中，通常使用大光圈（如f/1.4, f/1.8, f/2.8），以创造柔和的背景虚化（浅景深），突出人物。在团体照或需要更大景深的情况下，选择小一些的光圈（如f/5.6或更小）。

  4. ISO:

    • 设置: ISO决定了相机对光的敏感度。在光线充足的环境下，使用低ISO值（如ISO 100或200）。在光线较暗的环境下，可以适当提高ISO值以避免使用过慢的快门速度。

  5. 景深 (Depth of Field):

    • 设置: 景深受光圈大小、焦距和拍摄距离影响。在人像摄影中，常使用浅景深来突出人物，通过选择大光圈（小f值）来实现。

  6. 闪光灯 (Flash):

    • 设置: 如果在室内或光线不足的情况下，可以使用闪光灯来填充光线。避免直接对准模特，可以使用反光板或将闪光灯反射到天花板，创造更柔和的光线。

  7. 焦距 (Focal Length):

    • 设置: 在人像摄影中，中焦段镜头（如50mm, 85mm）是较理想的选择。这些焦距提供了较自然的视角和透视效果，同时允许足够的距离以与模特互动。

构图

基本的构图规则：最基本的规律便是将全景时的人物、 中近景时的脸部、或者大特写时的眼睛—当然还有任何你希望突出的人物体态和部 位，置于三分法画面中的四个交叉点之一。

  1. 点构图(1):

    • 解释: 这种构图方式将模特置于大景别的画面中，成为画面的唯一焦点。它通常用于组照中，作为一种视觉上的亮点，虽然不如特写那么直接，但它能在较大的环境中引起关注。
    • 例子: 想象一张照片，其中模特穿着醒目的服装站在一个开阔的草地上，周围是辽阔的天空和远处的山脉。模特虽然在画面中只占很小的一部分，但正是这种对比使得观众的注意力聚焦在模特身上。

  2. 对角线构图(2):

    • 解释: 这种构图通过利用画面的对角线，创造出一种动感和活力。它能让画面看起来更加稳定和饱满，同时打破传统的水平构图方式。
    • 例子: 设想一张上半身照片，模特的身体和视线沿着一条从画面一角延伸到另一角的对角线排列，创造了一种动态和深度的感觉。

  3. S形构图(3):

    • 解释: S形构图通过在图片中发现或创造弯曲的形态来增加动感和视觉吸引力。这种构图往往更注重于形状和线条。
    • 例子: 想象一张模特身体呈现优雅的S形弯曲的照片，背景可能是一条蜿蜒的小径或河流，这些线条与模特的姿势相呼应，创造了一种流动感。

  4. 三角形构图(4):

    • 解释: 通过模特的姿势创造出视觉上的三角形，这种构图方式可以带来均衡和稳定的感觉。
    • 例子: 可以想象一张照片，其中模特的头部位于顶部，而两只手臂向下伸展，形成一个倒置的三角形，这种姿势给人一种均衡和谐的视觉体验。

  5. 框架构图(5):

    • 解释: 这种构图利用前景或后景中的元素来“框”出主要的人物，增加画面的层次感和深度。
    • 例子: 想象一张模特站在一扇窗户前的照片，窗户的框架自然地引导观众的目光聚焦在模特身上，同时也为照片增加了额外的空间感。

  6. 打破构图(6):

    • 解释: 这不是一种具体的构图方式，而是一种鼓励摄影师打破常规和传统规则的观念。通过创新的构图，可以创造出独特和令人印象深刻的影像。
    • 例子: 这可以是一张非传统角度的照片，如模特的特写，只聚焦在眼睛或者嘴唇上，或者是一种极度夸张的角度拍摄，给人带来全新的视觉体验。

拍摄角度

  1. 平视: 适合拍摄正式场合的人像，如婚礼、毕业典礼等。这种角度可以让模特看起来更加自信和专注。缺乏特点
  2. 仰拍：显腿长，显身材；
  3. 俯拍：显脸小，显腿长，显身材；
  4. 侧拍：无限强化动感；
  5. 背面：神秘感

模特姿态处理

手的摆放:

其实无论哪种姿势，都主要是针对模特下 半身的变化而言的——实际上，模特的手在大 多数时候往往是结构画面的点睛元素。一般来 说，手的运用主要有两类：

  1. 一类是平衡画面。在这种情况下，手会作 为画面的重要组成部分存在，使人物姿态和画 面处在稳定的状态。
  2. 其次是塑造情绪，比如将手放在嘴上、头 部、胸口，所产生的情绪也是不尽相同的。

切记要比你的模特更早地进入拍摄状态，同时更迟地脱离出去，哪怕只是几秒钟时间 --> 只有这样，你才无愧于一个摄影师的称谓，要永远比你的模特思考得更多，坚持得更久。

光照

  1. 顺光：应当尽量选择黄昏和黎明的时候进行拍摄，避免太阳光线直射、光质 过硬而不易驾驭。
  2. 侧光：侧光拍摄容易形成强烈的对比明暗效果，使人物的轮廓鲜明立体；但是，如果光线过强，会造成人物的一侧过暗，这时候可以使用反光板来补光。
  3. 侧逆光：介于侧光和逆光之间的光照，亦是外拍中常用的光照方式。将太阳光作为轮廓光，凸显模特的发际和形体，而 用反光板或者闪光灯对模特的脸进行补光——这样的用光方式往往能强调模特忧伤、神秘的情绪。
  4. 逆光：
    • 一是以模特的脸部进行曝光，让背景过曝
    • 二是以背景进行曝光，凸显人物形态，用较低角度拍摄 模特的剪影
    • 在模特正面用反光板或其他手段进行补光

传统合影：

  1. 第一，作为摄影师，面对这么多人，首先得来个表名你身份的三脚架，以示重视。
  2. 第二，尽量用比f/8更小的光圈和比1/100s秒更快的快门进行拍摄，尽量采用连拍，保证总有一张是大家都没 闭眼的。
  3. 第三，如果合影人数较多，最好用中焦段镜头拍摄而在后期进行接片，避免广角造成的形变。
  4. 第四，如果合影人数实在让你有想偷懒用广角拍摄的想法，尽可能地让队伍两边的朋友往前站站，把队伍凑成 弧形，减少广角镜头形变造成的影响。
  5. 第五，除了一本正经地合影之外，如果场合不是特别严肃，还是让大家“耶”一张的好。

Q&A

  1. 拍照时快门速度太慢怎么回事？人像变虚了
  • 相机快门速度变慢是因为拍摄的物体本身颜色较暗或者是光线太暗；如果用的是光圈优先(曝光参数根据光圈自动调整)，为了达到正常的曝光，相机快门速度就会变慢，目的是为了得到正常影象；
  • 在光线不够的情况下可以提高感光度(ISO)，就可以得到正常曝光，此时相机快门速度就不会变慢。
  • 或者继续增大光圈
  • 或者开启闪光灯
  1. 如果你的镜头在55mm焦距下只能使用f/5.6光圈，你依然可以拍摄出不错的人像照片。虽然f/5.6的光圈不会提供极为浅的景深（即背景模糊），但它仍然可以在一定程度上突出人物。以下是一些建议来优化你的人像摄影：
    1. 调整你与被摄对象的距离：靠近你的拍摄对象可以增加背景模糊效果。记住，越接近你的拍摄对象，背景的虚化效果就越明显。
    2. 背景选择：选择一个简单、干净的背景。这样可以减少视觉干扰，即使背景不是非常模糊。
    3. 利用光线：使用合适的光线，如自然光或者柔光器，来更好地照亮你的拍摄对象。好的光线可以增强照片的质感和深度。
    4. 构图：认真考虑你的构图。利用规则三分法或者将被摄对象置于画面的有趣位置，可以创造更吸引人的照片。
    5. 焦距的调整：如果你的镜头允许变焦，尝试不同的焦距来看看哪个效果最好。长焦距通常更适合人像，因为它们提供了更自然的视角和适度的背景虚化。
