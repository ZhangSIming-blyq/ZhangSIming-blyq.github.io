在目前对风光摄影的定位上，大致可分为两个方向，分别为主观和客观的观看方法：

  1. 主观风景强调摄影师的主观感受，通过摄影师的视角来表现风景，这种风格的风景摄影作品，往往会有很强的个人风格；
  2. 客观风景强调摄影师的客观记录，通过摄影师的视角来记录风景，这种风格的风景摄影作品，往往会有很强的客观性。

配置参数

  1. ISO 设置:

    • 风光摄影通常需要使用最低的 ISO 设置（如 ISO 100 或 ISO 200），以保证画质的最大细腻度和最小的噪点。

  2. 焦段-光圈:

    • 广角镜头 (14mm - 35mm): 景观摄影: 为了获得从前景到背景的深广景深，选择 f/8 到 f/16。例如，24mm 配合 f/11 是广角风光摄影中的经典选择。
    • 标准焦距 (35mm - 70mm): 自然和城市景观: 如果目标是在不失去景深的情况下捕捉稍微紧凑的场景，可以使用 f/8 到 f/11。例如，50mm 配合 f/8。

注意，过小的光圈（如 f/22 或更小）可能会因衍射而降低图像的清晰度。

  1. 快门速度:

    • 根据光线条件和所需效果调整。在使用三脚架的情况下，可以使用较慢的快门速度，如几秒到几分钟，特别是在使用 ND 滤镜时。

  2. 白平衡:

    • 根据光源和拍摄环境调整，以恢复自然色彩。在户外，通常可以设置为日光或阴影模式; 也可以后期在手机调整。

  3. 曝光模式:

    • 手动模式（M或**光圈优先模式(A 或 Av)**更受青睐，因为它们提供了对曝光的更好控制。

  4. 文件格式:

    • 尽可能使用 RAW 格式拍摄，因为它提供了更多的后期处理灵活性和更高的图像质量。

风光摄影构图

1. 黄金分割构图

黄金分割构图遵循黄金比例（0.618），其中主要的景物放置在画面比例的约0.618处。这种构图方式自然而和谐，能有效吸引观众的注意力。在提供的示例中，您会看到主要的景物（如一棵树或一块岩石）被放置在符合黄金分割点的位置。

2. 线型构图

线型构图利用画面中的线条来引导观众的视线，创造深度和视角。在相应的示例中，您会看到像蜿蜒的河流或弯曲的道路这样的显著线条。

3. 对称构图

对称构图通过反映或重复的元素创造平衡和和谐。在提供的示例中，平静的湖面清晰地反映出山脉和天空，创造了一个视觉上吸引人且平衡的对称效果。

4. 分散式构图

分散式构图通过在画面中随机分布的元素（如树木、岩石、花朵）创建一种自然美感和随意性。这种构图方式看似没有明显的焦点，但在局部中可以发现内在的和谐。

5. 几何形构图

几何形构图使用如三角形这样的特定几何形状来增强视觉效果。在相应的示例中，山脉、树木或岩石被安排成三角形，创造了一个稳定而动态的构图。

6. 组合式构图

组合式构图结合了多种构图技巧，如引导线、几何形状和对称元素，以创造一个视觉上引人入胜且复杂的风景图像。相应的示例展示了这种复杂的场景。

7. 平衡构图

平衡构图通过在画面的两侧分布元素来创造视觉均衡感。在提供的示例中，如树木、山脉或建筑物等元素被均匀分布，使图像感觉和谐稳定。

8. 横竖构图

横竖构图考虑了画面的方向和形状。提供的示例中包含了两个图像：一个宽阔的横向风景，如山脉的全景，和一个垂直构图，强调如高大树木、瀑布或高山的纵深和垂直性。

拍风光要牢记

  1. 景深控制

20世纪30年代，由安塞尔·亚当斯和爱德华·韦斯顿等人创立的F64小组主张用极小的光圈进行大画幅摄影。 他们强调风光摄影画面必须在全范围内清晰

另一个方面，很多初学者喜欢用最大光圈拍摄所有事物，这显然是不适合的。多数镜头在最大光圈下的成像质 量并非最佳，甚至仅仅是差强人意。而广角镜头虽然拥有较大的景深，但在过大的光圈下仍然无法满足风光拍摄的 景深要求，因此应适当缩小光圈。

  1. 拍出风景中的动感: 长曝光+三角架

  2. 利用前景表现空间感

问题所在：画面显得很平

解决的方法也很简单。假设你在拍摄一片向日葵花海，你使用镜头的广角端直接拍摄，这时候就出现上面所说的问题了：没有重心，缺乏开阔感。这时候你只要挑一朵花作为前景（这朵花最好是较为出挑的），同样使用广 角端拍摄，这时候你会发现，背景里的大片向日葵瞬间有了活力，画面也一扫之前的平淡，变得动人起来。拍摄大 片房屋等场景时也是一样。找一个路牌或其他较为显眼的物体作为视觉重心，这样画面上就有了能让目光停留的地 方，并以该点为中心向周围辐散，欣赏到整张图的风景。如此，观看也就有了层次。

环境人像摄影

对于环境人像摄影，以下是几种常见的焦段和光圈组合，它们各自适用于不同的拍摄场景和目的：

  1. 广角镜头（24mm-35mm） + 大光圈（f/1.4 - f/2.8）:
    • 适用场景：城市街景、广阔自然风光。
    • 效果：背景广阔且有一定程度的虚化，突出人物同时展现环境。

  1. 标准镜头（50mm） + 中等光圈（f/4 - f/5.6）:
    • 适用场景：室内环境、小型户外活动。
    • 效果：平衡人物和背景的清晰度，适中地展现环境细节。

  1. 中远焦镜头（85mm-105mm） + 小光圈（f/8 - f/11）:
    • 适用场景：自然风光、城市建筑背景。
    • 效果：清晰展现人物和环境，适合全景式的环境人像。

  1. 超广角镜头（16mm-24mm） + 中等光圈（f/4 - f/5.6）:
    • 适用场景：极具视觉冲击的环境，如宽阔的天空、大型建筑。
    • 效果：极广的视野，同时保持人物和背景的清晰度。

  1. 长焦镜头（135mm-200mm） + 大光圈（f/2.8 - f/4）:
    • 适用场景：需要从远处拍摄，或希望将背景压缩至更小区域。
    • 效果：突出人物，背景虚化更显著，适合突出人物与特定环境元素的互动。
